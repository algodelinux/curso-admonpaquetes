Descarga de VirtualBox

¿De dónde descargo VirtualBox?

Podéis descargar VirtualBox desde la siguiente URL:

https://www.virtualbox.org/wiki/Downloads

Allí encontraréis paquetes de instalación para los diferentes sistemas operativos anfitriones. A continuación os dejamos algunos de los enlaces de descarga:

  • Windows hosts
  • OS X hosts
  • Linux distributions
    • Ubuntu 18,04
    • Ubuntu 22.04
    • Debian 11
    • Debian 12
  • Solaris hosts

La última versión disponible a día de hoy es la 7,0.

En el link Linux distributions encontraréis paquetes para instalar Virtualbox en diferentes distribuciones Linux. Descargad la versión que se corresponda con el sistema operativo en el que lo vais a instalar.
