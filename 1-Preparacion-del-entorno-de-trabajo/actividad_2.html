Actividad 2

Importar máquinas virtuales

Siguiendo el procedimiento descrito en la sección "Importar una máquina virtual":

  • Descarga la máquina virtual puppetserver.
  • Importa la máquina virtual puppetserver a partir del fichero de exportación puppetserver.ova. Utilizaremos esta máquina virtual como servidor puppet.
    • Esta máquina virtual tiene un usuario linex con contraseña linex y un usuario root con contraseña linex.
    • Una vez importada, configura la interfaz de red de la máquina virtual puppetserver en modo red NAT conectado a la red nat que creaste en la actividad anterior.
    • Configura la redirección de puertos como viste en el apartado Acceso ssh en modo red NAT para poder acceder a la máquina virtual puppetserver desde el anfitrión. Si quieres, utiliza el mismo puerto que usamos en dicho apartado: 2121.
    • No olvides agregar la línea "PermitRootLogin yes" al archivo de configuración /etc/ssh/sshd_config de la máquina virtual para que puedas acceder a ella como root con contraseña.
  • Descarga la máquina virtual bullseye.
  • Importa la máquina virtual bullseye a partir del fichero de exportación bullseye.ova. Al realizar la importación, renombra la máquina virtual como repositorio.
    • Esta máquina virtual tiene un usuario linex con contraseña linex y un usuario root con contraseña linex. 
    • Una vez importada, configura la interfaz de red de la máquina virtual repositorio en modo red NAT conectado a la red nat que creaste en la actividad anterior.
    • Configura la redirección de puertos como viste en el apartado Acceso ssh en modo red NAT para poder acceder a la máquina virtual puppetserver desde el anfitrión. Naturalmente, deberás usar otro puerto. Por ejemplo, el 2122.
    • No olvides agregar la línea "PermitRootLogin yes" al archivo de configuración /etc/ssh/sshd_config de la máquina virtual para que puedas acceder a ella como root con contraseña.
